# 前端框架用来测试的后端服务
===============

## 概述

>主要适用技术栈：

+ 后端框架：ThinkPHP 5.0.6
+ 前端MVVM框架：自制简单的框架

>不仅适用于管理后台或管理系统开发，且广泛适用于B/S架构的项目开发。

> ThinkPHP的运行环境要求PHP5.4以上。

>> 详细开发文档参考 [ThinkPHP5完全开发手册](http://www.kancloud.cn/manual/thinkphp5)
